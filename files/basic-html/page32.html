管道 3：國立臺灣師範大學僑生先修部


               如錯過「個人申請」及「聯合分發」，或學科測驗成績未達入讀大學門檻，可透過國立臺灣師範大學僑生先修部（簡
               稱僑先部），修讀為期一年的先修課程( 大學預備教育 )。僑先部為臺灣唯一辦理僑生大學先修教育之學府，創設於
     第         1955 年秋季，至今六十多年，歷史悠久。僑先部旨在協助僑生完成大學先修教育，強化學生基本學科能力，打好基礎，
     二         以凖備大學課程，並以結業成績分發至臺灣各大學院校。
     章
               僑先部歷屆結業生，均順利分發至各公私立大學院校就讀，且於大學畢業後，不論在學術上繼續深造、研究或回僑居
     赴         地開創事業，均有極其優異之表現與貢獻。
     臺
     升
     學          課程教學
     管          參照大學院系劃分，依升學意願分為一、二、三類組，另因華語文需要，加設特別輔導班( 修業一年，再升入先修班 )
     道          等四大類別，實施分組教學。學生一律住校，並有完善的生活照顧及獎學金制度。
     介
     紹          就學費用估計
                含食宿、書籍、制服、保險等，一年約新臺幣 80,000 元（約港幣 20,000 元）
                （實際費用以僑先部公布為準）

                入學方式
                有「自願免試申請」和「聯合分發」兩種方式，本章節主要介紹「自願免試申請」，「聯合分發」詳情請參閱前頁。

                報名資格

                1. 持澳門永久居民身分證
                2. 最近連續居留境外6 年以上（申請就讀大學醫學、牙醫及中醫學系者須滿8 年；所稱境外，指臺灣以外之國家或地區。）
                3. 中學畢業 ( 包括應屆中六畢業生 )
                報名資料

                1. 資料檢核表 1 份
                2. 申請表一式 4 份
                3. 澳門永久性居民身分證副本4 份
                4. 中學畢業證書副本1 份（應屆畢業生可先檢送在學證明書或學生證副本，並註明就讀年級）
                5. 中學最後二年成績單1 份
                6. 香港或澳門居民身分聲明書1 份
                ＊以上資料以副本繳交者，須備正本核驗
