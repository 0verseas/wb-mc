管道 4：升讀研究所 ( 碩士班 / 博士班 )


            提升自我競爭力，升讀研究所是最好的選擇。同學可依未來工作所需能力選擇系所，運用研究所資源增強
            知識，加深加廣相關技能。在進入職場時，因具備專業知能優勢，而相對有較優渥的薪資或升遷機會，翻                                                            第二章

            轉人生。


            報名資格
                                                                                                                      赴臺升學管道介紹
            1. 持澳門永久居民身分證
            2. 最近連續居留境外 6 年以上 ( 所稱境外，指臺灣以外之國家或地區。)
            3. 學歷
                (1) 經臺灣教育部認可之澳門當地大學 / 碩士畢業，具有學士 / 碩士學位，其學歷證件及歷年成績證明並經驗證屬實者。
                (2) 在澳門以外大學畢業具有學士 / 碩士學位，其學歷證件須經學歷完成地之臺灣駐外館處 ( 如台北經濟文化辦事處 ) 驗證。
                (3) 臺灣各大學畢業且取得學士 / 碩士學位者。
